Recipe App
This is a standalone command line application that allows users to create, scale, reset and clear recipes. 

Compiling the Application
To compile the application, follow these steps:
1.Open Visual Studio
2.Open the solution file (RecipeApp.sln) in the root directory of the project.
3.Navigate the Build Solution by selecting the Build menu on the task bar.

Running the Application
To run the application, follow these steps:
1.Open Visual Studio
2.Navigate the file menu on the task bar 
3.Click the solution file (RecipeApp.sln) in the root directory of the project.
4.Pressing Start or alternatively F5 to run the application

Using the Application
The user is prompted to enter the quantity of ingredients and the number of steps for the recipe when the program launches. 
The recipe is then created after the user is prompted to enter the specifics for each ingredient and step.
Four options are available to the user after the recipe is created: 
scaling the recipe by a factor of 0.5, 2, or 3, resetting the quantities to their original values, 
clearing the recipe to enter a new one, or closing the program.
